# Change Log

## [1.0.0] - 2024-01-09

### Added
- Initial release of Templ Syntax Highlight extension
- Syntax highlighting for templ template files
- Support for component definitions
- Support for embedded CSS blocks
- Support for embedded JavaScript blocks
- Support for HTML elements and attributes
- Support for Go expressions within templates
- Code snippets for common templ patterns
- Auto-completion for templ keywords and HTML tags
- Hover information for templ-specific constructs
- Basic document formatting
- Custom icons for templ files
- Dark theme optimized for templ syntax
- Example templ file with comprehensive syntax demonstrations

### Features
- Language recognition for `.templ` files
- IntelliSense support
- Status bar indicator
- Command palette integration
- Comprehensive documentation and examples

### Known Issues
- Basic formatting may need refinement for complex templ files
- Some edge cases in syntax highlighting may not be perfectly handled
